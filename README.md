# Yaks Java Client API

The Java API for Yaks, based on the zenoh-java API.

## Building
Requirements:
 - Java >= 8
 - Apache Maven >= 3.6.0

To build:
```mvn clean install```

To generate the Javadoc and a ZIP file for release in assembly/target:
```mvn -Prelease clean install```
